# 隐私说明

快密生成器是一款本地优先的随机密码生成器 Chrome 扩展。

## 数据收集

本扩展不收集、出售、共享或上传任何用户个人信息。

## 密码数据

- 生成的密码只显示在当前扩展弹窗中，或在用户主动操作时填入当前网页。
- 扩展不会把密码发送到任何服务器。
- 关闭弹窗后，当前显示的密码不会作为配置写入本地存储。

## 密码历史

本扩展会在用户本地浏览器中保存最近的密码使用历史，用于帮助用户找回近期复制或自动填充过的密码。

保存的历史信息包括：

- 生成后的密码。
- 使用动作，例如复制或自动填充。
- 使用时间。
- 自动填充时所在网站的域名，例如 `accounts.example.com`。

本扩展不会保存完整 URL、页面路径、查询参数、页面内容或表单内容。密码历史仅保存在用户自己的浏览器本地存储 `chrome.storage.local` 中，不会上传到任何服务器，不会出售或共享给第三方。

用户可以在扩展界面中随时清空历史记录。

## 本地配置

扩展会使用 `chrome.storage.local` 保存以下配置偏好：

- 密码长度。
- 是否包含大写字母。
- 是否包含小写字母。
- 是否包含数字。
- 是否包含特殊符号。
- 是否排除易混淆字符。
- 语言设置。

这些配置仅保存在用户自己的浏览器本地，用于下次打开扩展时恢复设置。

## 网络请求

本扩展不主动发起网络请求，也不依赖远程服务生成密码。

## 权限说明

扩展使用的 Chrome 权限：

- `storage`：用于在本地保存用户配置偏好、语言设置和密码使用历史。
- `activeTab`：仅在用户主动点击自动填充时访问当前标签页。
- `scripting`：仅在用户主动点击自动填充时，把本地生成的密码填入当前页面输入框。

扩展声明 `<all_urls>` 的 content script，用于在网页密码输入框旁显示本地填充按钮。扩展不会读取、保存或上传页面内容。

## 联系与更新

如果后续扩展增加新功能并涉及新的数据处理方式，应同步更新本文档与 Chrome Web Store 隐私声明。
