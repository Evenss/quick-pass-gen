# Chrome Web Store 上架文案草稿

## 插件名称

快密生成器

## 简短描述

一款简洁安全的随机密码生成器，支持自定义长度、字符类型和一键复制。

## 详细描述

快密生成器是一款轻量级 Chrome 扩展，帮助你快速生成安全随机密码。

主要功能：

- 自定义密码长度，支持 8 到 64 位。
- 可选择大写字母、小写字母、数字和兼容特殊符号。
- 支持排除易混淆字符，降低手动输入出错概率。
- 使用浏览器安全随机数 API 生成密码。
- 一键复制生成结果。
- 自动在本地保存配置偏好。
- 不保存密码历史，不上传任何密码数据。

快密生成器适合用于注册网站账号、更新账号密码、生成临时强密码等场景。所有密码均在本地生成，不依赖远程服务器。

## 分类建议

- 生产力工具
- 工具

## 隐私摘要

本扩展不收集用户个人信息，不保存生成过的密码，不向服务器上传数据。扩展仅使用 `chrome.storage.local` 在本地保存用户配置偏好。
