# Chrome Web Store 上架文案草稿

## 插件名称

快密生成器

## 简短描述

一款简洁安全的随机密码生成器，支持自定义长度、字符类型和一键复制。

## 详细描述

快密生成器是一款轻量级 Chrome 扩展，帮助你快速生成安全随机密码。

主要功能：

- 自定义密码长度，支持 8 到 32 位。
- 可选择大写字母、小写字母、数字和兼容特殊符号。
- 启用字母时，密码首位优先使用字母，减少网站兼容问题。
- 支持排除易混淆字符，降低手动输入出错概率。
- 使用浏览器安全随机数 API 生成密码。
- 一键复制生成结果。
- 支持在本地保存最近使用历史，方便找回近期复制或填充过的密码。
- 历史记录仅保存在用户本机浏览器中。
- 自动填充历史只记录网站域名，不记录完整 URL、页面路径、查询参数、页面内容或表单内容。
- 不上传任何密码数据。
- 用户可以随时清空历史记录。

快密生成器适合用于注册网站账号、更新账号密码、生成临时强密码等场景。所有密码均在本地生成，不依赖远程服务器。

## 隐私摘要

本扩展会在本地保存配置偏好、语言设置和最近密码使用历史。密码历史仅保存在用户自己的 Chrome 本地存储中；自动填充历史只保存网站域名，不保存完整 URL 或页面内容。扩展不会上传任何密码数据。
