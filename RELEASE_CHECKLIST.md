# 发布检查清单

## 代码检查

- [ ] 确认 `manifest.json` 中版本号已更新。
- [ ] 确认插件名称、描述符合当前发布版本。
- [ ] 确认没有提交 `dist/`、`node_modules/` 等生成目录。
- [ ] 确认没有保存任何真实密码、测试账号或敏感信息。

## 测试检查

- [ ] 运行 `npm test` 并确认通过。
- [ ] 在 Chrome `chrome://extensions/` 中加载已解压扩展并手动测试。
- [ ] 测试密码长度滑块。
- [ ] 测试字符类型开关。
- [ ] 测试排除易混淆字符。
- [ ] 测试重新生成按钮。
- [ ] 测试复制按钮。
- [ ] 关闭并重新打开插件，确认配置已恢复。

## 打包检查

- [ ] 运行 `npm run pack:chrome`。
- [ ] 确认生成 `dist/quick-pass-gen.zip`。
- [ ] 确认 zip 包中包含 `manifest.json`、popup 文件、密码逻辑和图标。

## Chrome Web Store 检查

- [ ] 上传 `dist/quick-pass-gen.zip`。
- [ ] 填写简短描述。
- [ ] 填写详细描述。
- [ ] 上传截图。
- [ ] 填写隐私声明。
- [ ] 确认权限说明只包含必要权限。
- [ ] 提交审核。
