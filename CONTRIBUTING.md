# 贡献指南

感谢关注快密生成器。这个项目目前定位为轻量、本地优先的 Chrome 随机密码生成器。

## 开发原则

- 保持插件简单、快速、易用。
- 不保存生成过的密码历史。
- 不新增不必要的 Chrome 权限。
- 不引入远程密码生成服务。
- 密码生成逻辑应保持可测试。
- UI 修改应优先保证 popup 小尺寸下的可读性。

## 本地检查

提交前请运行：

```bash
npm test
```

如果修改了打包相关文件，也请运行：

```bash
npm run pack:chrome
```

## 代码风格

- 使用原生 JavaScript、HTML 和 CSS。
- 密码生成核心逻辑放在 `password.js`。
- popup 交互逻辑放在 `popup.js`。
- 新增密码规则时，请同步添加测试。

## 隐私与安全

任何新功能都不能默认保存生成过的密码。如果确实要新增敏感功能，需要先更新隐私说明、README 和发布文案。
